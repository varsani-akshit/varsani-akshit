## what i'm building

### [Trodo](https://trodo.ai)
AI-native observability for agents. most analytics tools were built
for page views and click events — they're blind to what actually
happens inside an AI agent. prompt sequences, tool call chains, retry
loops, failure modes. that's the whole game now and nothing was 
watching it properly. that's what Trodo fixes.

actively building this full time.
